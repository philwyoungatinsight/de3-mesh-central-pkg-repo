# tg-scripts/mesh-central-pkg/mesh-central

MeshCentral remote-management server scripts.

| Script | Purpose |
|--------|---------|
| [`install/`](install/README.md) | Install MeshCentral server, apply AMT TLS patches, configure admin account. |
| [`update/`](update/README.md) | Enroll all `role_mgmt_group_*` hosts as managed devices; configure Intel AMT credentials. |
